A #pl118 do café
Originally shared by Eduardo LuísSociedade Protectora dos Pequenos Comerciantes de Café - SPPCC

Vou abrir uma sociedade para proteger os pequenos comerciantes de cápsulas de café.
A actual sociedade hoje vive em torno das cápsulas de café e o comportamento padrão da sociedade é reutilizar as cápsulas de café. Já viram as perdas destes pequenos comerciantes de café quando vendem as cápsulas uma vez e depois os consumidores as reutilizam com café das grandes superfícies?

Está provado que este é o comportamento padrão como se pode constatar no estudo feito no pacote de açucar
ali no café da D. Ermelinda.

A taxa será aplicada aos seguintes produtos:

Talheres - Sempre que compram talheres, compram certamente colheres; ora estas irão servir para mexer o café.

Açucar - Sempre que o consumidor compra açucar, será para usar no café.

Chávenas - Sempre que o consumidor compra chávenas, só pode ser para usar café.

Máquinas de café - Não restam dúvidas, serão utilizadas somente para tirar cafés com cápsulas reutilizadas - Está provado.

A distribuição dessas taxas será:
50% para a SPPCC
30% para as pequenas agências que representam a SPPCC
20% para distribuir pelos pequenos comerciantes de café.

Obrigado.
